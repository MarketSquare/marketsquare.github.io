Community Code Of Conduct

MarketSquare is dedicated to providing a harassment-free experience for everyone, regardless of gender, gender identity and expression, sexual orientation, disability, physical appearance, body size, age, race, nationality, or religion. We do not tolerate harassment of participants in any form.

This code of conduct applies to all MarketSquare spaces, including but not limited to community organizaion, projects, teams, and repositories, both online and off. Anyone who violates this code of conduct may be sanctioned or expelled from these spaces at the discretion of the MarketSquare Maintainers.

Some MarketSquare spaces may have additional rules in place, which will be made clearly available to participants. Participants are responsible for knowing and abiding by these rules.

Join organization

To join the community's MarketSquare Organization, please open a ticket. Note: you must have a GitHub account to open a ticket (and further actions). You can create one for free here.

New repository

You can create new repositories in the square as you wish. Keep in mind that you will lose administrator access to them a few minutes later. If you inadvertently create a repository in the MarketSquare, please open a ticket.

Other issues

For all other issues, please open a ticket.

Disabling notification spam

By default you are subscribed to all new MarketSquare github repository notifications. Unless you disable this yourself soon your inbox will contain only GitHub spam.

  • Go to https://github.com/watching
  • Select what to watch or Uncheck Automatically watch. But attention: latter will disable watch for all organizations.

Transferring repository ownership to MarketSquare

How to move your repository under MarketSquare shared ownership. About transfering the ownership of repositories see GitHub instructions regarding transfers. You can do this yourself with GitHub's *Transfer Ownership* button 'if the repository belongs to your personal account.

For repositories part of any GitHub organization (and organization only) you need the organization admin permissions to perform the transfer. Admin in the third party organization must be also MarketSquare member.
